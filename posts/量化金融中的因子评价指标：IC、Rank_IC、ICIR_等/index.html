量化金融中的因子评价指标：IC、Rank IC、ICIR 等

在量化投资里，做出一个新因子（Factor）只是开始，真正难的问题是：这个因子到底能不能预测未来收益？业界为此发展出了一套评价体系，核心就是 IC（Information Coefficient）、Rank IC、ICIR、IC Decay，以及分层收益、多空收益、胜率这几个指标。本文按从简单到深入的顺序把它们过一遍，包括含义、算法和实际用法。

什么是因子（Factor）

先明确一下因子是什么：一个用来预测股票未来收益的数值特征。常见的例子有 PE、PB、市值、动量（Momentum）、波动率（Volatility）、RSI、MACD、北向资金流入、新闻情绪，甚至机器学习模型直接输出的一个预测分数——只要是给每只股票打一个数、并且这个数被认为跟未来收益有关，就算因子。

比如取 factor = PE 的倒数，市场里的每只股票都能算出一个对应的因子值。接下来要验证的是：因子值越大的股票，未来是不是真的涨得更多？后面所有的指标，其实都是在从不同角度回答这一个问题。

IC（Information Coefficient）

IC 是最经典的因子评价指标，衡量的是因子值和未来收益率之间的 Pearson 相关系数。设 fif_ifi​ 为第 iii 支股票的因子值，rir_iri​ 为它的未来收益率，则：

IC=Corr(f,r)IC = Corr(f, r)IC=Corr(f,r)

其中 Corr 就是 Pearson 相关系数。

示例

假设有四只股票：

股票因子值下月收益
A1.58%
B0.94%
C0.3-2%
D-0.5-6%

因子越大、收益越高，两者同向变化，这种情况下 IC 会接近 1。反过来，如果收益变成：

股票因子值下月收益
A1.5-6%
B0.9-3%
C0.35%
D-0.59%

IC 就会接近 -1——因子和收益完全反着走，说明这个因子反向使用反而有效。如果因子和收益压根没关系，IC 会趋近于 0，也就意味着这个因子没有预测能力。

如何理解 IC 数值

很多人第一次看到 IC 时都会有点意外：一个被认为”优秀”的因子，IC 往往只有 0.03 左右，远不是想象中的 0.8、0.9。这是因为股票市场的噪声极大，任何单一因子能解释的部分本来就有限。经验上的大致区间是：

IC一般评价
< 0.02较弱
0.02 ~ 0.05较好
0.05 ~ 0.10优秀
> 0.10非常优秀，较少见

所以评价一个因子时，不要拿”相关系数应该很高”这种直觉去套，真实金融市场里几乎不会出现这种数值。

Rank IC

IC 用的是 Pearson 相关系数，有个明显的缺点：容易被异常值带偏。比如因子值是 1, 2, 3, 100000，最后这个离群值可能会把整体相关系数完全带偏。

因此量化投资里更常用的其实是 Rank IC，它用的是 Spearman 秩相关系数——不比较数值本身的大小，只比较排序是否一致。举个例子，因子排序和未来收益排序都是：

股票排名
A1
B2
C3
D4

哪怕实际收益具体是 20%、15%、5%、-3% 这种不均匀的分布，只要排序一致，Rank IC 依然会接近 1。这也是它相比 IC 的优势：对异常值更稳健，不要求变量间是线性关系，只关心排序能力本身。国内大多数券商和公募基金研究因子时，更偏好用 Rank IC 也是这个原因。

IC Time Series

IC 不是算一次就完事的，通常每天（或每个调仓周期）都会重新计算一遍，比如：

Day1   0.04
Day2   0.06
Day3  -0.01
Day4   0.03
...

这样就得到了一条 IC 时间序列，之后可以进一步看平均 IC、IC 的波动、IC 的稳定性——这几个统计量往往比某一天的单次 IC 更能说明问题。

ICIR（Information Coefficient Information Ratio）

ICIR 关心的是另一个维度：因子的预测能力是否稳定。定义为：

ICIR=IC‾σ(IC)ICIR = \frac{\overline{IC}}{\sigma(IC)}ICIR=σ(IC)IC​

也就是 IC 的均值除以 IC 的标准差。举例来说，因子 A 的 IC 序列是 0.05, 0.05, 0.04, 0.05，波动很小，ICIR 就会很高；因子 B 是 0.20, -0.15, 0.18, -0.20，虽然平均下来看起来也不差，但正负剧烈波动，ICIR 反而很低。

简单说，IC 衡量的是预测能力本身，ICIR 衡量的是这份能力稳不稳定——一个长期稳定发挥的因子，通常比偶尔表现惊艳的因子更值得信赖。

IC Decay

一个好因子不只是能预测明天，还得知道这份预测能力能撑多久。做法是计算不同预测周期下的 IC，比如：

IC(1天)   = 0.08
IC(5天)   = 0.05
IC(10天)  = 0.03
IC(20天)  = 0.01

随着预测周期拉长，IC 逐渐衰减，这个现象就叫 IC Decay。它能帮忙确定因子的有效期、最佳调仓周期，以及这个因子更适合做高频策略还是低频策略。

分层收益（Group Return）

IC 只能说明因子和收益存在相关性，但投资者真正关心的是：按这个因子去买股票，到底赚不赚钱？验证方式是把所有股票按因子值排序后分组，比如 1000 支股票平均分成 10 组（Decile），再对比 Top 10% 和 Bottom 10% 未来的收益表现。

理想情况下，收益会随着分组从 Top10% 到 Bottom10% 单调下降——如果确实如此，说明这个因子的区分能力很好，这个性质也叫 Monotonic（单调性）。

多空收益（Long-Short Return）

在分层的基础上可以再进一步，构造一个经典的测试组合：买入 Top10%、卖空 Bottom10%，组合收益就是 Long-Short Return。如果长期来看 Long-Short Return > 0，说明这个因子不只是统计意义上显著，还具备实际的可交易价值——这也是很多量化研究报告里最看重的验证方式之一。

胜率（Win Rate）

另一个常用指标是胜率。比如过去 200 个交易日里有 160 天 IC > 0，胜率就是 80%。相比只看平均 IC，胜率反映的是因子是否长期保持有效——大多数时间都能站对方向，比平均值高但偶尔翻车更有价值。

总结

量化投资中，一个新的 Alpha 因子通常会经过如下评价流程：

一个因子
      │
      ▼
计算未来收益
      │
      ├── IC（线性相关性）
      ├── Rank IC（排序相关性）
      ├── IC Mean（平均预测能力）
      ├── IC Std（预测能力波动）
      ├── ICIR（预测能力稳定性）
      ├── IC Decay（预测能力持续时间）
      ├── Group Return（分层收益）
      ├── Long-Short Return（多空收益）
      └── Win Rate（有效比例）

概括一下各指标的角色：IC 判断因子是否具有预测能力；Rank IC 更稳健，是最常用的指标；ICIR 判断预测能力是否稳定；IC Decay 判断预测能力能持续多久；Group Return 看收益分层效果；Long-Short Return 看实际可交易价值；Win Rate 看长期有效性。

实际做因子研究时，很少会只依赖单一指标，通常是 Rank IC、ICIR、分层收益、多空收益几个维度一起看，才能判断一个因子是否真正具有 Alpha 能力。
