神经网络入门：从神经元到反向传播

这是一篇面向初学者的神经网络笔记。目标不是一上来堆公式，而是从“神经元到底是什么”开始，循序渐进地理解：前向传播、损失函数、梯度、学习率、参数更新，以及两层神经网络的反向传播。

  • 神经元：一个带参数的计算单元
  • 参数：权重 www 和偏置 bbb

  • 梯度：告诉参数往哪边改

1. 为什么要学神经网络

刚接触神经网络时，很容易被一堆名词绕晕：神经元、权重、偏置、激活函数、前向传播、反向传播、梯度下降……

但如果把这些概念拆开看，会发现神经网络的核心并不神秘：

  • 它本质上是一个带参数的函数；
  • 它通过前向传播算出预测结果；
  • 再通过反向传播计算参数应该如何调整；
  • 最终目标不是让参数本身变小，而是让损失函数（loss）变小。

这篇文章就按照这个思路，一步一步讲清楚神经网络。

2. 神经元到底是什么

很多初学者都会问：

  • 神经元是一个值吗？
  • 神经元是一组公式吗？
  • 神经元之间是映射关系吗？

更准确的说法是：

神经元本身是一个计算单元，也可以理解成一个带参数的函数；而一次前向传播后，神经元会输出一个值。

单个神经元通常写成：

a=σ(w⋅x+b)a = \sigma(w \cdot x + b)a=σ(w⋅x+b)

其中：

  • xxx：输入
  • www：权重（weight）
  • bbb：偏置（bias）
  • σ\sigmaσ：激活函数
  • aaa：神经元输出

所以可以把三件事区分开：

  1. 神经元：一个计算规则
  2. 参数 w,bw,bw,b：这个规则里的可学习变量
  3. 输出 aaa：一次计算得到的数值

如果用程序员更容易接受的方式来理解，单个神经元有点像这样：

def neuron(x, w, b):
    z = w * x + b
    a = activation(z)
    return a

这里：

  • neuron(...) 这个函数本身，像“神经元”
  • w, b 是这个函数里的参数
  • return a 返回的才是神经元的输出值

3. 神经网络是什么

如果说一个神经元是一个简单函数，那么神经网络就是：

很多简单函数组合起来，形成一个更复杂的函数。

从数学角度，可以写成：

y=f(x;θ)y = f(x; \theta)y=f(x;θ)

其中：

  • xxx 是输入
  • yyy 是输出
  • θ\thetaθ 表示网络里所有参数（所有层的 WWW 和 bbb）

所以神经网络本质上是在做一件事：

学习一个从输入到输出的映射关系。

4. 权重 www 和偏置 bbb 从哪里来

这是刚学神经网络时最关键的问题之一。

答案是：

不是人手动写死的，而是在训练过程中学出来的。

通常流程如下：

  1. 先随机初始化参数 w,bw,bw,b
  2. 把训练数据输入网络
  3. 网络做前向传播，得到预测值
  4. 计算预测值和真实值之间的误差（loss）
  5. 通过反向传播计算每个参数的梯度
  6. 用梯度下降更新参数
  7. 重复很多轮，参数逐渐变好

所以：

  • 初始的 w,bw,bw,b 是随机的
  • 后续的 w,bw,bw,b 是通过训练不断更新出来的

5. 前向传播：神经网络如何“算结果”

前向传播（Forward Propagation）可以概括为一句话：

用当前参数，沿着网络从前往后算出预测结果。

以最简单的单个神经元为例：

y^=wx+b\hat{y} = wx + by^​=wx+b

假设：

  • x=2x=2x=2
  • w=3w=3w=3
  • b=1b=1b=1

那么预测结果就是：

y^=3⋅2+1=7\hat{y} = 3 \cdot 2 + 1 = 7y^​=3⋅2+1=7

如果真实值是：

y=10y=10y=10

那么模型就预测错了。

前向传播做的事情只有两步：

  1. 计算预测值 y^\hat{y}y^​
  2. 配合真实值 yyy 计算损失 LLL

6. 损失函数：模型到底错了多少

损失函数（Loss Function）用来衡量：

当前模型预测得有多差。

一个常见的损失函数是平方误差：

L=(y^−y)2L = (\hat{y} - y)^2L=(y^​−y)2

比如刚才：

  • y^=7\hat{y}=7y^​=7
  • y=10y=10y=10

则：

L=(7−10)2=9L = (7-10)^2 = 9L=(7−10)2=9

损失越大，说明预测越差；损失越小，说明预测越准。

这里一定要记住一个关键点：

神经网络训练的目标，不是让 w,bw,bw,b 变小，而是让 loss 变小。

参数只是工具，loss 才是最终优化目标。

7. 反向传播：神经网络如何“改参数”

如果前向传播是在“算结果”，那么反向传播（Backpropagation）就是在“改参数”。

它的核心作用是：

计算损失函数对每个参数的偏导数（梯度），从而知道这些参数应该往哪个方向调整。

比如我们想知道：

∂L∂w,∂L∂b\frac{\partial L}{\partial w}, \quad \frac{\partial L}{\partial b}∂w∂L​,∂b∂L​

它们表示：

  • 如果改动一点点 www，loss 会怎么变
  • 如果改动一点点 bbb，loss 会怎么变

因为计算是从输出层开始，再一层层往回传，所以叫“反向传播”。

8. 梯度到底是什么

梯度可以先从一维的导数来理解。

假设 loss 是参数 www 的函数：

L=L(w)L = L(w)L=L(w)

那么导数 dLdw\frac{dL}{dw}dwdL​ 表示：

在当前点，www 每变化一点，loss 大概怎么变化。

它可能是正数、负数，或者 0：

  • 若梯度为正：增大 www 会让 loss 增大，因此应该让 www 变小
  • 若梯度为负：增大 www 会让 loss 变小，因此应该让 www 变大
  • 若梯度为 0：当前位置可能已经到极值附近

这就是为什么偏导数是有正负的。

它不是单纯表示“大小”，还表示“方向”。

9. 学习率是什么

学习率通常记作 η\etaη（eta），有些资料也写成 α\alphaα。

它表示：

每次参数更新时，走多大一步。

学习率太大：

  • 可能直接跨过最优点
  • 导致震荡甚至发散

学习率太小：

  • 虽然方向对
  • 但收敛非常慢

所以：

  • 梯度决定方向
  • 学习率决定步长

10. 参数更新公式是怎么来的

最经典的参数更新公式是：

w=w−η∂L∂ww = w - \eta \frac{\partial L}{\partial w}w=w−η∂w∂L​ b=b−η∂L∂bb = b - \eta \frac{\partial L}{\partial b}b=b−η∂b∂L​

很多人第一次看到会问：为什么一定是这个形式？

10.1 直觉理解

因为梯度指向的是“函数上升最快的方向”，而我们想让损失下降，所以要往梯度的反方向走。

这就是减号的来源。

10.2 一阶近似理解

在当前参数点附近，用一阶泰勒展开近似：

L(w+Δw)≈L(w)+∂L∂w⋅ΔwL(w + \Delta w) \approx L(w) + \frac{\partial L}{\partial w} \cdot \Delta wL(w+Δw)≈L(w)+∂w∂L​⋅Δw

要让 loss 下降，就希望后面这一项尽量为负。

于是自然想到令：

Δw=−η∂L∂w\Delta w = -\eta \frac{\partial L}{\partial w}Δw=−η∂w∂L​

代入可得：

L(w+Δw)≈L(w)−η(∂L∂w)2L(w + \Delta w) \approx L(w) - \eta \left(\frac{\partial L}{\partial w}\right)^2L(w+Δw)≈L(w)−η(∂w∂L​)2

只要学习率合适，这一项就会让 loss 下降。

所以更新公式就成了：

wnew=w−η∂L∂ww_{new} = w - \eta \frac{\partial L}{\partial w}wnew​=w−η∂w∂L​

这就是梯度下降（Gradient Descent）的基本思想。

11. 单层神经元的前向与反向

我们继续用一个最简单的例子来打通逻辑。

设：

  • 输入 x=2x=2x=2
  • 真实值 y=10y=10y=10
  • 初始参数 w=3,b=1w=3, b=1w=3,b=1
  • 模型：y^=wx+b\hat{y} = wx+by^​=wx+b
  • 损失：L=(y^−y)2L=(\hat{y}-y)^2L=(y^​−y)2

11.1 前向传播

先算预测值：

y^=3⋅2+1=7\hat{y}=3\cdot 2+1=7y^​=3⋅2+1=7

再算 loss：

L=(7−10)2=9L=(7-10)^2=9L=(7−10)2=9

11.2 反向传播

先求：

∂L∂y^=2(y^−y)=2(7−10)=−6\frac{\partial L}{\partial \hat{y}} = 2(\hat{y}-y)=2(7-10)=-6∂y^​∂L​=2(y^​−y)=2(7−10)=−6

又因为：

y^=wx+b\hat{y}=wx+by^​=wx+b

所以：

∂y^∂w=x=2\frac{\partial \hat{y}}{\partial w}=x=2∂w∂y^​​=x=2 ∂y^∂b=1\frac{\partial \hat{y}}{\partial b}=1∂b∂y^​​=1

于是根据链式法则：

∂L∂w=∂L∂y^⋅∂y^∂w=−6⋅2=−12\frac{\partial L}{\partial w} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial w} = -6 \cdot 2 = -12∂w∂L​=∂y^​∂L​⋅∂w∂y^​​=−6⋅2=−12 ∂L∂b=∂L∂y^⋅∂y^∂b=−6⋅1=−6\frac{\partial L}{\partial b} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial b} = -6 \cdot 1 = -6∂b∂L​=∂y^​∂L​⋅∂b∂y^​​=−6⋅1=−6

如果学习率 η=0.1\eta=0.1η=0.1，那么更新参数：

w=3−0.1(−12)=4.2w = 3 - 0.1(-12)=4.2w=3−0.1(−12)=4.2 b=1−0.1(−6)=1.6b = 1 - 0.1(-6)=1.6b=1−0.1(−6)=1.6

再做一次前向传播：

y^=4.2⋅2+1.6=10\hat{y}=4.2\cdot 2+1.6=10y^​=4.2⋅2+1.6=10

可以看到，loss 直接降到了 0。

12. 两层神经网络：从前向到反向一步一步看

为了更贴近真正的神经网络，我们再看一个最小的两层网络：

  • 输入层：1 个输入 xxx
  • 隐藏层：1 个神经元
  • 输出层：1 个神经元

12.1 前向传播公式

隐藏层：

z1=w1x+b1z_1 = w_1x + b_1z1​=w1​x+b1​ h=σ(z1)h = \sigma(z_1)h=σ(z1​)

输出层：

z2=w2h+b2z_2 = w_2h + b_2z2​=w2​h+b2​ y^=σ(z2)\hat{y} = \sigma(z_2)y^​=σ(z2​)

损失函数：

L=12(y^−y)2L = \frac{1}{2}(\hat{y}-y)^2L=21​(y^​−y)2

其中 sigmoid 函数为：

σ(z)=11+e−z\sigma(z)=\frac{1}{1+e^{-z}}σ(z)=1+e−z1​

它的导数是：

σ′(z)=σ(z)(1−σ(z))\sigma'(z)=\sigma(z)(1-\sigma(z))σ′(z)=σ(z)(1−σ(z))

12.2 给定数值

设：

  • x=1x=1x=1
  • y=1y=1y=1
  • w1=0.5w_1=0.5w1​=0.5
  • b1=0b_1=0b1​=0
  • w2=1w_2=1w2​=1
  • b2=0b_2=0b2​=0

12.3 前向传播计算

隐藏层线性部分：

z1=0.5⋅1+0=0.5z_1 = 0.5 \cdot 1 + 0 = 0.5z1​=0.5⋅1+0=0.5

隐藏层输出：

h=σ(0.5)≈0.622459h = \sigma(0.5) \approx 0.622459h=σ(0.5)≈0.622459

输出层线性部分：

z2=1⋅0.622459+0=0.622459z_2 = 1 \cdot 0.622459 + 0 = 0.622459z2​=1⋅0.622459+0=0.622459

最终输出：

y^=σ(0.622459)≈0.650778\hat{y} = \sigma(0.622459) \approx 0.650778y^​=σ(0.622459)≈0.650778

损失：

L=12(0.650778−1)2≈0.060978L = \frac{1}{2}(0.650778-1)^2 \approx 0.060978L=21​(0.650778−1)2≈0.060978

12.4 反向传播计算

首先求输出层误差：

∂L∂y^=y^−y=0.650778−1=−0.349222\frac{\partial L}{\partial \hat{y}} = \hat{y}-y = 0.650778-1 = -0.349222∂y^​∂L​=y^​−y=0.650778−1=−0.349222 ∂y^∂z2=y^(1−y^)≈0.227267\frac{\partial \hat{y}}{\partial z_2}=\hat{y}(1-\hat{y})\approx 0.227267∂z2​∂y^​​=y^​(1−y^​)≈0.227267

所以：

δ2=∂L∂z2=∂L∂y^⋅∂y^∂z2≈−0.079363\delta_2 = \frac{\partial L}{\partial z_2} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z_2} \approx -0.079363δ2​=∂z2​∂L​=∂y^​∂L​⋅∂z2​∂y^​​≈−0.079363

再求输出层参数梯度：

∂L∂w2=δ2⋅h≈−0.079363⋅0.622459≈−0.049402\frac{\partial L}{\partial w_2} = \delta_2 \cdot h \approx -0.079363 \cdot 0.622459 \approx -0.049402∂w2​∂L​=δ2​⋅h≈−0.079363⋅0.622459≈−0.049402 ∂L∂b2=δ2≈−0.079363\frac{\partial L}{\partial b_2} = \delta_2 \approx -0.079363∂b2​∂L​=δ2​≈−0.079363

接着把误差传回隐藏层：

∂L∂h=δ2⋅w2=−0.079363⋅1=−0.079363\frac{\partial L}{\partial h} = \delta_2 \cdot w_2 = -0.079363 \cdot 1 = -0.079363∂h∂L​=δ2​⋅w2​=−0.079363⋅1=−0.079363 ∂h∂z1=h(1−h)≈0.235004\frac{\partial h}{\partial z_1} = h(1-h) \approx 0.235004∂z1​∂h​=h(1−h)≈0.235004

所以隐藏层误差为：

δ1=∂L∂z1=∂L∂h⋅∂h∂z1≈−0.018651\delta_1 = \frac{\partial L}{\partial z_1} = \frac{\partial L}{\partial h} \cdot \frac{\partial h}{\partial z_1} \approx -0.018651δ1​=∂z1​∂L​=∂h∂L​⋅∂z1​∂h​≈−0.018651

隐藏层参数梯度：

∂L∂w1=δ1⋅x=−0.018651⋅1=−0.018651\frac{\partial L}{\partial w_1} = \delta_1 \cdot x = -0.018651 \cdot 1 = -0.018651∂w1​∂L​=δ1​⋅x=−0.018651⋅1=−0.018651 ∂L∂b1=δ1=−0.018651\frac{\partial L}{\partial b_1} = \delta_1 = -0.018651∂b1​∂L​=δ1​=−0.018651

12.5 结果理解

这个例子最关键的不是数字本身，而是你会发现：

  • 输出层误差先算出来
  • 再通过链式法则传给隐藏层
  • 每一层都通过“后一层误差 × 当前层局部导数”来得到自己的梯度

这就是反向传播的本质。

13. 为什么说反向传播的核心是链式法则

以两层神经网络为例，loss 对隐藏层参数并不是直接依赖，而是经过这样一条链：

w1→z1→h→z2→y^→Lw_1 \rightarrow z_1 \rightarrow h \rightarrow z_2 \rightarrow \hat{y} \rightarrow Lw1​→z1​→h→z2​→y^​→L

所以必须一层一层往回传：

∂L∂w1=∂L∂y^⋅∂y^∂z2⋅∂z2∂h⋅∂h∂z1⋅∂z1∂w1\frac{\partial L}{\partial w_1} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z_2} \cdot \frac{\partial z_2}{\partial h} \cdot \frac{\partial h}{\partial z_1} \cdot \frac{\partial z_1}{\partial w_1}∂w1​∂L​=∂y^​∂L​⋅∂z2​∂y^​​⋅∂h∂z2​​⋅∂z1​∂h​⋅∂w1​∂z1​​

这就是链式法则在神经网络里的直接体现。

所以可以说：

反向传播并不是新的数学，而是链式法则在多层复合函数中的系统化应用。

14. 神经网络里每个单元到底是“值”还是“公式”

这是一个非常容易混淆的问题。

更准确的表述应该是：

  • 神经元本身是一个计算单元
  • 它内部有公式
  • 它在一次传播后会输出一个值

所以：

  • 若从“结构”上看：神经元像一个函数模块
  • 若从“运行结果”上看：神经元会产出一个数值
  • 若从“整层”上看：一层神经网络可以看成一个向量到向量的映射

这三种说法都不是完全错，关键是你在说的是哪个层面。

15. 一个适合初学者的整体理解框架

学到这里，可以把神经网络压缩成下面这套理解框架：

15.1 神经元是什么

神经元是一个带参数的函数：

a=σ(w⋅x+b)a = \sigma(w \cdot x + b)a=σ(w⋅x+b)

15.2 神经网络是什么

神经网络是很多神经元组成的多层函数：

y=f(x;θ)y = f(x;\theta)y=f(x;θ)

15.3 前向传播是什么

前向传播就是：

用当前参数，从输入一路计算到输出，再计算 loss。

15.4 反向传播是什么

反向传播就是：

从 loss 开始，利用链式法则，逐层计算 loss 对各参数的梯度。

15.5 梯度下降是什么

梯度下降就是：

用梯度告诉我们该往哪边改参数，并通过学习率控制改多少。

更新公式为：

θ←θ−η∇θL\theta \leftarrow \theta - \eta \nabla_\theta Lθ←θ−η∇θ​L

16. 这只是开始：后面还应该学什么

理解了本文这些内容，已经打下了神经网络最重要的基础。但如果想继续深入，后面还有几个必学主题：

  1. 激活函数为什么必要

    • 如果没有激活函数，多层网络会退化成线性变换

  2. 矩阵形式的前向与反向传播

    • 实际代码里几乎都用矩阵表示，而不是逐个神经元去算

  3. 常见损失函数

    • 均方误差、交叉熵等

  4. 常见优化器

    • SGD、Momentum、RMSProp、Adam

  5. 深层网络中的问题

    • 梯度消失、梯度爆炸、初始化、正则化等

17. 总结

神经网络乍看很复杂，但核心逻辑其实很朴素：

  • 用参数定义一个函数
  • 用前向传播算结果
  • 用损失函数衡量错误
  • 用反向传播求梯度
  • 用梯度下降更新参数
  • 重复这个过程，让模型越来越好

如果要用一句话概括：

神经网络 = 多层可学习函数；训练神经网络 = 不断调整函数参数，让预测误差越来越小。

当把这条主线真正理解之后，再去看卷积神经网络、Transformer、注意力机制，就不会只停留在“背概念”的层面，而是能从“它到底在算什么、为什么能训练”这个角度真正看懂。
